Умный юзербот для рассылки объявлений в Telegram чаты с поддержкой:
- 🤖 Уникализации текста через ИИ (OpenAI)
- ⏰ Реалистичных паттернов активности (утро, обед, вечер)
- 📊 Лимитов постов в день на чат
- 🔄 Копи-пасты (каждый 5-й чат)
- 🚫 Защиты от FLOOD_WAIT и блокировок

## ⚙️ Требования

- .NET 8.0+
- SQLite
- Telegram API ключи (my.telegram.org)
- OpenAI-совместимый API для ИИ (опционально)

## 🚀 Установка

1. **Клонируйте репозиторий:**
```bash
git clone <repo>
cd TestApp
```

2. **Настройте `appsettings.json`:**
```json
{
  "Telegram": {
    "ApiId": 12345678,
    "ApiHash": "your_api_hash",
    "PhoneNumber": "+77000000000",
    "Password2Fa": "optional_2fa_password"
  },
  "Ai": {
    "AiApiKey": "sk-...",
    "AiModelName": "gpt-3.5-turbo",
    "AiBaseUrl": "https://api.openai.com/v1/"
  },
  "ConnectionStrings": {
    "DefaultConnection": "Data Source=userbot.db"
  }
}
```

3. **Запустите приложение:**
```bash
dotnet run
```

При первом запуске потребуется авторизация в Telegram (номер + SMS код).

## 📝 Команды в "Избранное"

### `/add_chat {ID} {название} [N]`
Добавить чат в рассылку
```
/add_chat -1001234567890 MyGroup 5
         └─ ID         └─ Название └─ Макс 5 постов/день
```

### `/del_chat {ID}`
Удалить чат из рассылки
```
/del_chat -1001234567890
```

### `/set_text {текст}`
Установить текст объявления
```
/set_text Привет! Проверьте наше предложение...
```

### `/status`
Показать статус рассылки:
- Список всех чатов
- Текущий прогресс (X из Y постов за день)
- Текущий текст объявления

### `/logs`
Показать последние 10 отправок и ошибок

### `/help`
Справка по всем командам

## 🧠 Логика уникализации текста

1. **Каждый 5-й чат** получает новый текст от ИИ
2. **Следующие 4 чата** получают копи-пасту (быстро отправляется)
3. **Время набора**:
   - ИИ: 3-8 секунд (зависит от % изменения текста)
   - Копи-паста: 0.5-1.2 секунды

Пример:
```
Чат 1: ✨ ИИ → 7 сек → "Переписанный текст"
Чат 2: 📋 Копи → 0.8 сек → "Переписанный текст"
Чат 3: 📋 Копи → 0.6 сек → "Переписанный текст"
Чат 4: 📋 Копи → 0.9 сек → "Переписанный текст"
Чат 5: 📋 Копи → 0.7 сек → "Переписанный текст"
Чат 6: ✨ ИИ → 5 сек → "Новая версия текста"
...
```

## 🔧 Настройки на чат

Для каждого чата можно установить:
- **PostsPerDay** - макс постов в день (0 = без лимита)
- **SlowModeSeconds** - минимум секунд между отправками (по умолчанию 60)
- **IsActive** - включена ли рассылка

Счётчик автоматически сбрасывается в полночь UTC.

## 📊 База данных

SQLite с таблицами:
- **TargetChats** - целевые чаты для рассылки
- **AdvertisingTemplates** - шаблоны объявлений
- **ExecutionLogs** - логи всех отправок и ошибок

Миграции выполняются автоматически при запуске.

## 🛡️ Обработка ошибок

| Ошибка | Действие |
|--------|----------|
| FLOOD_WAIT | Глобальная пауза на указанное время |
| RPC ошибка | Логирование, продолжение работы |
| Чат не найден | Логирование ошибки, попытка позже |

## 🐛 Troubleshooting

**Проблема: "FLOOD_WAIT"**
- Решение: Дождитесь указанного времени, затем перезапустите

**Проблема: "Чат не найден"**
- Проверьте, что ID чата правильный (формат: -1001234567890)
- Убедитесь, что бот состоит в этом чате

**Проблема: "Ошибка ИИ"**
- Проверьте API ключ и базовый URL
- При ошибке использует исходный текст (fallback)
