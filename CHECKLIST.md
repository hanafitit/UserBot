## 💾 База данных

- [ ] SQLite база создана автоматически (`userbot.db`)
- [ ] Миграции применены при первом запуске

## 🧪 Тестирование локально

```bash
# Сборка
dotnet build

# Запуск
dotnet run
```

При запуске:
- [ ] Приложение просит номер телефона (если не в конфиге)
- [ ] Приложение просит код из SMS
- [ ] Показывает данные пользователя
- [ ] Планировщик запустился
- [ ] Команды слушаются в "Избранном"

## 📱 Тестирование команд в Telegram

1. **Перейдите в "Избранное"** (Сохранённые сообщения)

2. Отправьте `/help`
   - [ ] Должен ответить справка
   - [ ] Ответ идёт с "печатает..."

3. Отправьте `/add_chat -123456789 TestGroup 2`
   - [ ] Ответ: чат добавлен для рассылки

4. Отправьте `/set_text Привет тестовое сообщение`
   - [ ] Ответ: шаблон установлен

5. Отправьте `/status`
   - [ ] Видно список чатов
   - [ ] Видно текущий шаблон

6. Отправьте `/logs`
   - [ ] Видны последние логи
